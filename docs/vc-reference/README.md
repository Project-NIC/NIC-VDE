# Volkov Commander — archived reference

A single preserved copy of the original **Volkov Commander** sources by
**Vsevolod V. Volkov**, kept here only as a **backup reference** for Volkov
Data's UI behavior (two-pane layout, F1–F10 keys, dialogs, status bar) — so the
behavior reference is at hand and doesn't have to be hunted down online.

- `archives/vc49909.zip` — the original source archive of Volkov Commander
  **4.99.09** (the most complete tree). x86 assembly; **reference only**, not
  ported code. Unzip it if you need to read a specific routine
  (`VCPANELS.INC`, `VCFUNC.INC`, `VCKEYB.INC`, `VCDIALOG.INC`, …).
